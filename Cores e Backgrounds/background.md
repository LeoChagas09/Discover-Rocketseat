## Background

- Define um fundo para o nosso elemento
- Sua área de atuacao é a caixa toda
- por padrão, é transparente

### Exemplos

- Usar cores sólidas
- Usar imagens
- Controlar
    - a posicao das imagens,
    - se elas se repetem ou nao
    - o tamanho delas na caixa
- Usar cor e imagem juntas
- Usar cor gradiente
