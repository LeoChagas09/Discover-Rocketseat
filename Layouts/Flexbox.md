## Flexbox

* Nos permite posicionar os elementos dentro da caixa.
* Controle em uma dimensao (horizontal ou vertical)
* Alinhamento, direcionamento, ordernar e tamanhos

```css
div.parent {
    diplay: flex;
}
```

## flex-direction

* Qual a direcao do flex: horizontal ou vertical
* row | column

## Alinhamento

* justify-content
* align-items
